移动端产品经理

岗位职责Job Description

1、负责石头理财移动平台（iOS或Android）的产品设计和产品规划；

2、协调各部门资源，推进产品迭代；

3、充分了解用户需求，结合移动端特性设计产品，不仅限于Web功能的延伸，需具备移动端的特色和亮点；

4、负责部分跨平台产品功能的调研和设计；

5、关注产品数据和用户反馈，提炼用户需求，指导产品设计；

6、完成产品迭代以及用户活跃增长等目标；

任职要求Qualifications

1、相关专业大专及以上学历；

2、3年以上移动互联网经验，曾独立负责过产品版本的迭代；

3、拥有清晰的产品设计思路，对行业有全面了解和独到认识，拥有整体把握产品的能力；

4、优秀的逻辑思维能力和沟通表达能力，严谨负责的工作态度，优秀的学习能力；

5、能够输出高质量的产品原型和需求文档；

6、拥有互联网金融或支付类产品经验者优先，有团队管理经验者优先；

请发送您的简历至hr@stlc.cn，标题请注明所申请职位。

运维工程师

岗位职责Job Description

1、负责石头理财平台自动化监控，发现问题及时处理；

2、负责石头理财网站开发测试环境的维护工作；

3、负责石头理财网站托管服务器及网络规划、管理、维护以及日常运维工作；

4、协助编写相关运维技术文档，完善运维制度和流程；

5、领导安排的其他工作；

任职要求Qualifications

1、有1-2年大型网站系统运维相关工作经验,优秀应届生也可；

2、熟悉Linux基础系统管理和应用（Nginx/Tomcat/Jboss/Mysql/Oracle/Redis等）的安装、配置、优化；

3、熟悉shell脚本，以及掌握sed/awk工具；

4、熟悉SVN GIT代码控制工具的配置使用 以及持续集成工具Hudson工具的使用；

5、熟悉Mysql结构，掌握常用的SQL语句，具备分析造成mysql瓶颈的能力；

6、有均衡技术、应用集群技术、数据库高可用和读写分离、分布式存储维护经验者优先。

请发送您的简历至hr@stlc.cn，标题请注明所申请职位。

移动端测试工程师

岗位职责Job Description

1、根据产品规范编写测试计划，设计测试数据和测试用例；

2、完成对产品的集成测试与系统测试，对产品的功能、性能及其他方面的测试负责；

3、结合项目测试需求，完成自动化脚本、工具开发；

4、负责项目自动化相关系统的设计和开发；

任职要求Qualifications

1、相关专业大专及以上学历，计算机相关专业优先；

2、2年以上手机客户端测试经验，或1年以上客户端开发经验；

3、有手机客户端自动化测试经验优先；

4、熟悉java/C++/Python/Php/Perl其中一种编程语言；

5、具备数据库（MySQL）操作及查询语句编写经验；

6、熟悉各类手机系统，如Android，iOS等；

7、有较强的学习能力，对新技术可以快速上手，并敢于使用新方法或技术解决问题；

8、具有良好的沟通、团队协作、创新及一定的抗压能力。

请发送您的简历至hr@stlc.cn，标题请注明所申请职位。
